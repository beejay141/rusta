# Blog API Documentation

This directory contains documentation for the Blog API example service.

## Files

- **POSTMAN.md** - Complete Postman documentation with all endpoints, sample data, and testing workflows
- **Blog API.postman_collection.json** - Ready-to-import Postman collection

## Quick Start

1. Start the services:

   ```bash
   docker compose up -d
   ```

2. Import the Postman collection from `Blog API.postman_collection.json`

3. Follow the testing workflow in [POSTMAN.md](POSTMAN.md)
